# aws-lambda-container-images

This repository supports [Part 1](https://blog.curiousdev.io/aws-lambda-container-images-part-1-aws-base-image-for-lambda) and [Part 2](https://blog.curiousdev.io/aws-lambda-container-images-part-2-aws-os-only-image-for-lambda) of the AWS Lambda Container Images blog series. We will look at mechanisms to build out containerized Lambda functions using several options:

* [AWS base image for Lambda](./aws-base-images/README.md)
* [AWS OS-only image for Lambda](./aws-os-only-images/README.md)

The repository is self-contained; each example can be run independently.
