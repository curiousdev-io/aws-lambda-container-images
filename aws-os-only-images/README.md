# AWS OS-Only Base Images

AWS OS-only base images contain an Amazon Linux distribution and the [runtime interface emulator](https://github.com/aws/aws-lambda-runtime-interface-emulator/). The `lambda/provided` image can be found in the [Amazon ECR Public Gallery](https://gallery.ecr.aws/lambda/provided).
